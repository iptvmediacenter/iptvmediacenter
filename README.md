## 📺 About IPTV Mediacenter

**IPTV Mediacenter** is a high-performance, premium media player designed for mobile screens and fully optimized for Android TV (Leanback) devices.

### Key Features:
*   **D-Pad Support**: Fully optimized focus states, remote control click events, and tactile animations designed for television remotes.
*   **ExoPlayer Engine**: Low-latency, high-performance decoding for HLS and Transport Stream (TS) live broadcasts.
*   **Cloud Sync**: Secure synchronization of favorites and "Continue Watching" playback history via Firebase cloud infrastructure.
*   **Empty Out-Of-The-Box**: A completely neutral player. Users must load their own playlists (M3U or Xtream Codes) legally.
